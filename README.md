# README
# Doutorado_Populismo

Repositório de desenvolvimento computacional da pesquisa **Populismo na Era de Narciso: Carisma, Identificação e Organização Política na América Contemporânea**.

Este projeto reúne notebooks, scripts, bases processadas, tabelas e documentação utilizados na etapa empírica da pesquisa de doutorado.

## Objetivo

O objetivo do repositório é garantir a organização, rastreabilidade e reprodutibilidade das análises empíricas desenvolvidas ao longo da pesquisa.

A investigação articula dados individuais, institucionais e partidários provenientes de bases como:

* World Values Survey (WVS);
* Varieties of Democracy (V-Dem);
* V-Party.

## Estrutura do repositório

```text
notebooks/      Notebooks analíticos da pesquisa
scripts/        Funções auxiliares e ferramentas do projeto
dados/          Bases processadas e integradas
resultados/     Tabelas e produtos analíticos exportados
PROJECT.md      Documento mestre do projeto
CHANGELOG.md    Registro cronológico das etapas concluídas
```

## Estado atual

Até o momento, foram concluídas as etapas de preparação da base WVS, construção de indicadores, análise fatorial exploratória, fatores de segunda ordem e identificação dos perfis afetivos.

A etapa em desenvolvimento é a integração entre WVS e V-Dem.

## Observação sobre dados brutos

As bases brutas não são versionadas neste repositório, em razão de tamanho, organização e reprodutibilidade. O repositório mantém apenas dados processados, produtos intermediários e resultados necessários à continuidade da análise.
